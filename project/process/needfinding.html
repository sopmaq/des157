User Needs

  1. Needs to be able to find and understand all possible interactive choices.
  2. Needs to understand the purpose of the interface.
  3. Needs to know how to input data via the microphone.
  4. Neds to have access to a way of inputting microphone data.
  5. Needs to not be overwhelmed by color or type.
  6. Needs to understand the relationship between mic data and visual data.
  7. Needs to not be too distracted on breathing.
