User Needs

  1. Needs to be able to find all possible interactive choices.

  2. Needs to understand the purpose of the interface.

  3. Needs to understand the relationship between mic data and visual data.

  4. Needs to know how to input data via the microphone.

  5. Needs to have access to a way of inputting microphone data.

  6. Needs to not be overwhelmed by color or type.

  7. Needs to not be distracted by or conscious of their breathing.

  8. Needs to be able to easily navigate to more options.

  9. Needs to be able to stay focused on the visualizer and not get distracted.

  10. Needs to feel less anxious than when started, if using anxiety visualizer to relieve anxiety.
